Box Model

In web development, the CSS box model refers to how HTML elements are modeled in browser engines and how dimension of those HTML elements are derived from CSS properties. It is a fundamental concept for the composition of HTML webpages. The guidelines of the box model are described by web standards World Wide Web Consortium (W3C) specifically the CSS Working Group. For much of late-1990s and early 2000s there had been non-standard compliant implementations of the box model in the mainstream browsers. With the advent of CSS2, which introduced the box-sizing property, the problem had mostly been resolved.

Box Model Pratybos

Sukurkite 40x40 pikselių dydžio kvadratus (bet kokios spalvos) kurie turėtų:

  • 10px margin iš viršaus ir apačios
  • 10px margin iš kairės
  • 10px margin iš visų pusių, išskyrus dešinės
  • padding iš visų pusių po 10px
  • 10px margin viršuje, 10px padding iš kairės ir apačios. Uždėkite mė sienelę
  • 10px visose pusėsė, 10px padding viršuje ir apačioje
  • 10px keirėje pusėsė, 10px padding dešinėje. Uždėkite mė sienelę.
